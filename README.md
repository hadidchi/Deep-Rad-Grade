# Deep-Rad-Grade
This is the code used to generate CNN models for analysis of dynamic contrast-enhanced breast MRIs and the classification of intermediate risk tumors (Nottingham Histological Grade 2) into lower-risk and higher-risk subgroups.

Link to paper: https://doi.org/10.1007/s00330-025-12208-6

To access the data, please see https://www.cancerimagingarchive.net/collection/duke-breast-cancer-mri/
* Images can be downloaded via NBIA Data Retriever
* Also download Annotation Boxes and Clinical and Other Features
